# 🌙 NOX – Landing Page Club Nocturno

Landing page prototipo para el club nocturno **Nox**, ubicado en San Rafael, Mendoza.  
El objetivo principal es ofrecer una experiencia visual atractiva y moderna, permitiendo a los usuarios explorar el ambiente del lugar antes de visitarlo.

---

## 🚀 Tecnologías utilizadas

- **Next.js** – Framework principal
- **React** – UI
- **v0** – Generación rápida del prototipo base
- **Imágenes 360°** – Experiencia inmersiva del espacio

---

## 🧪 Características

- Diseño moderno orientado a nightlife
- Navegación simple y directa
- Integración de vista **360°** para recorrer el club
- Enfoque visual (imágenes y ambientación)
- Prototipo rápido para validación de concepto

---

## 📸 Experiencia 360°

Uno de los diferenciales de esta landing es la integración de imágenes en **360 grados**, permitiendo:

- Explorar el interior del club
- Generar mayor confianza en potenciales clientes
- Simular la experiencia real del lugar
